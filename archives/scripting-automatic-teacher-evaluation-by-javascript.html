实在是不爽，于是想着自己写个快速评教脚本，顺便练习一下久违的 javascript
于是动手，用 chrome 在线调试功能来写
之前只是稍微见过别人调试，没有自己深究过这玩意
现在发现 chrome 的 console 还真是挺强大的一个工具

这个控制台其实就是一个调试 javascript 的控制台
